What to do:


If you're using a shared hosting, sometimes you can select which version of PHP and MySQL to use. Check your account Control Panel to see if you can change the MySQL version to 5.6 or higher. If not, maybe you need to open a ticket with them (or migrate to a hosting company that likes updated software).


If you're on a VPS, dedicated server or a local server (localhost):
- If you're using a package manager (like apt-get or homebrew): upgrade your repository base (e.g. apt-get upgrade) and update the MySQL package using its corresponding command.
- If you're running bundled software stacks like MAMP, WampServer or XAMPP, or used the graphical MySQL installer, maybe just upgrading your stack is enough. Download the new version from the vendor website;
- If your MySQL was compiled from source: read the MySQL documentation to see how to upgrade.

Anything we can do?


If any of these solutions work, try to get some help opening an Issue in our GitHub repository.
